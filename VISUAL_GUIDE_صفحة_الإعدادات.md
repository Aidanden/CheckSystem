## 🎨 الألوان والتصميم:

### 1. المربع الأزرق (شرح خط MICR):
```
┌────────────────────────────────────────────────────┐
│ 📘 ترتيب البيانات (من اليسار لليمين):           │
│                                                    │
│ [رقم التسلسل 9 أرقام] [الرقم التوجيهي]          │
│ [رقم الحساب 15 رقم] [نوع الصك: 01 أفراد/02 شركات]│
│                                                    │
│ مثال: 000000001 1100000001 100012345678901 01    │
└────────────────────────────────────────────────────┘
```
- **الموقع**: فوق إعدادات خط MICR مباشرة
- **اللون**: خلفية زرقاء فاتحة `bg-blue-50`
- **الحدود**: `border-blue-200`
- **النص**: `text-blue-800`

### 2. المربع الأخضر (تفصيل المكونات):
```
┌────────────────────────────────────────────────┐
│ 📋 تكوين خط MICR:                             │
│                                                │
│ • التسلسل: 000000001 (9 أرقام)               │

│ • النوع: 01 (أفراد) أو 02 (شركات)            │
```

- **الموقع**: تحت معاينة الشيك

- **الحدود**: `border-green-200`

## 🔍 نظام الإحداثيات:

```
        X (أفقي) →→→→→→→
Y  (0,0) ┌────────────────────────────┐
│    │                            │
│    │    اسم الفرع (117.5, 10)  │
عمودي   │                            │
↓    │              التسلسل (200, 18)│
│    │                            │
│    │                            │
↓    │                            │
     │ صاحب الحساب (20, 70)       │
     │ خط MICR (117.5, 80)        │
     └────────────────────────────┘
```

**ملاحظة:**
- X = 0 → الطرف الأيسر للشيك
- Y = 0 → الطرف الأعلى للشيك
- X = checkWidth → الطرف الأيمن
- Y = checkHeight → الطرف الأسفل

## 💡 نصائح للاستخدام:

### 1. للمحاذاة:
- **يسار**: النص يبدأ من X
- **وسط**: النص يتوسط عند X
- **يمين**: النص ينتهي عند X

### 2. للتسلسل (عادة على اليمين):
```
X: 200 ملم (قرب الحافة اليمنى)
Y: 18 ملم (قريب من الأعلى)
المحاذاة: يمين
```

### 3. لخط MICR (عادة في الأسفل):
```
X: 117.5 ملم (منتصف الشيك 235÷2)
Y: 80 ملم (قرب الأسفل)
المحاذاة: وسط
```

- ✅ اضبط قيمة واحدة
- ✅ راقب المعاينة
- ✅ اضبط أكثر إذا لزم الأمر
- ✅ احفظ
